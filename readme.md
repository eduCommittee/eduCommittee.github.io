# MMC Onboarding Platform

This is the **Moroccan Mathematical Community (MMC) Onboarding Platform**, designed to allow MMC members, especially new comers and idle members, to:

- Discover MMC educational project committee (onboarding):
  - Mission Statement
  - Projects
  - Members
  - Prerequisite Skills
  - What You Will Learn
  - Work Culture
  - Documentation
  - FAQ
- Submit their applications to join the committee (recruitment)
